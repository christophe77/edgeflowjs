# i18n API

Internationalization for multi-language kiosks. See [guides/i18n.md](../guides/i18n.md) for setup and usage.

## Install

```bash
pnpm add @edgeflowjs/i18n
```

## Core exports

```ts
import {
  createI18n,
  type I18n,
  type Locale,
  type Translations,
} from "@edgeflowjs/i18n";
```

## React exports

```ts
import { I18nProvider, useI18n, useT } from "@edgeflowjs/i18n/react";
```

## Types

### Locale

```ts
type Locale = string;  // e.g. "en", "fr"
```

### Translations

```ts
type Translations = Record<string, string | Record<string, string>>;
```

Supports flat keys (`"screen.idle.ready"`) or nested objects.

### I18n

```ts
type I18n = {
  locale: Locale;
  setLocale(locale: Locale): void;
  t(key: string, params?: Record<string, string | number>): string;
  formatDate(date: Date, options?: Intl.DateTimeFormatOptions): string;
  formatNumber(n: number, options?: Intl.NumberFormatOptions): string;
};
```

## createI18n

```ts
function createI18n(opts: {
  defaultLocale: Locale;
  translations: Record<Locale, Translations>;
  onLocaleChange?: (locale: Locale) => void;
}): I18n;
```

Creates an i18n instance. `onLocaleChange` is optional; when provided, it is called when `setLocale` is invoked. Use it to trigger React re-renders when the locale changes.

## t(key, params?)

Translates a key, with optional param substitution.

- **Fallback:** If the key is missing in the current locale, falls back to `defaultLocale`.
- **Params:** `t("greeting", { name: "User" })` replaces `{{name}}` in the translation string.

## formatDate

Formats a date using `Intl.DateTimeFormat` with the current locale.

```ts
i18n.formatDate(new Date());
i18n.formatDate(new Date(), { dateStyle: "short" });
```

## formatNumber

Formats a number using `Intl.NumberFormat` with the current locale.

```ts
i18n.formatNumber(1234.56);
i18n.formatNumber(1234.56, { style: "currency", currency: "EUR" });
```

## React components and hooks

### I18nProvider

```tsx
function I18nProvider(props: { i18n: I18n; children: ReactNode }): JSX.Element;
```

Wraps the app to provide the i18n instance to descendants.

### useI18n

```ts
function useI18n(): I18n;
```

Returns the i18n instance. Throws if used outside `I18nProvider`.

### useT

```ts
function useT(): (key: string, params?: Record<string, string | number>) => string;
```

Returns the translation function bound to the current i18n instance. Use for translating strings in components.
